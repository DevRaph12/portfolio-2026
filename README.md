# 🌐 Raphael Almeida | Front-end Portfolio

This is my personal portfolio, built to showcase my core strengths in Vanilla JavaScript and Web Accessibility (WCAG) as I prepare for a transition into the React ecosystem.

## 🚀 Key Features

-   **Native i18n (Internationalization):** A custom-built language switcher (PT-BR/EN-US) using HTML5 `data-attributes` and Vanilla JS.
-   **Theme Management:** Dark and Light modes with persistent state logic.
-   **Web Accessibility (WCAG):** Implementation of semantic HTML, ARIA labels, and `aria-live` for screen readers.
-   **Responsive Design:** A mobile-first approach ensuring a perfect experience on screens as small as 320px.

## 🛠️ Technical Stack

-   **HTML5:** Structured for SEO and screen readers.
-   **CSS3:** Modern architecture using CSS Variables, Flexbox/Grid, and Glassmorphism effects.
-   **Vanilla JavaScript:** Logic for theme toggling, multi-language support, and clipboard API for emails.
